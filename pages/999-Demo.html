Hello!

netzine-cms is an experimental content management system of reasonable design defaults, dreamt for use in creative academic contexts.

It assumes that its users are already busy authoring their next brilliant metatextual opus, and don't want to be bothered with writing code for common navigation tasks. It aims to be unobtrusive, lightweight, and as drop-in-place as possible. It provides a self-populating menu (see bottom right) and assumes some decent default behaviors for handling common file types.

Getting Started

  1. Download netzine-cms and place it on your web server.
  2. Place files into the pages directory.

That's it!

Supported Filetypes

netzine-cms currently has handlers written for common web formats (.html, .php), most major image formats (.jpg, .gif, .png, and others), as well as most common programming formats (.js, .py, .rb, ...).

to see how netzine-cms handles different filetypes, check out the demos in the menu.
